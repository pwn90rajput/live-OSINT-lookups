Open OSINT Monitor

Live, client-side lookups for any domain or brand using free, no-login sources. Works entirely on GitHub Pages.

Tip: You can scan with Domain, Brand, or both. Brand-only mode checks many TLDs (e.g., dell.in, dell.ru, dell.xyz) and CT logs for lookalikes.

Sources: Cloudflare & Google DoH (DNS), RDAP, ipwho.is (GeoIP/ASN), crt.sh (CT), sonar.omnisint.io (subdomains). No API keys required; modest rate-limits apply.
